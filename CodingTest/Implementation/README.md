Implementation Algorithm
=============

* Implementation Algorithm은 머릿속으로 구상한 알고리즘을 Source Code로 바꾸는 과정을 말한다.
* 여기서는 완전 탐색, 시뮬레이션 유형을 모두 "Implementation"유형으로 묶어서 다루고 있으나 몇몇 책에서는 완전 탐색, 시뮬레이션 유형을 별도로 다루는 경우도 있다.
  * 완전 탐색 : 모든 경우의 수를 하나도 빠짐없이 전부 다 계산하는 해결 방법
  * 시뮬레이션 : 문제에서 제시한 알고리즘을 한 단계씩 차례대로 직접 수행
