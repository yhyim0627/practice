코딩테스트 준비용 공간
--------------------
* 이곳은 "이것이 취업을 위한 코딩 테스트다 with 파이썬(나동빈 저)" 책을 기반으로 코딩테스트 공부를 정리하는 공간입니다.
  
* 코딩테스트 준비를 위해 공부한 내용에 대해 다음과 같은 주요 알고리즘 이론을 분류별로 구분하여 정리하고 관련된 실전 알고리즘 문제를 풀어봅니다.
  * Greedy(그리디)
  * Implementation(구현)
  * DFS/BFS(깊이 우선 탐색/너비 우선 탐색)
  * Sorting(정렬)
  * Binary Search(이진 탐색)
  * Dynamic Programming(다이나믹 프로그래밍)
  * Shortest Path(최단 경로)
  * Graph Theory(그래프 이론)

* 코딩테스트에서는 해당 프로그램의 실행 시간 제한과 메모리 용량 제한이 주어지며, 실험적인 방법으로 실행 시간을 계산해보는 코드는 ex01.py와 같습니다.
